# 顔生成ルール（v1・首から上）

「気持ち悪く見えない」ことを最優先に、古典的な人相比率を骨格にしている。
数値は `src/render/proportions.js`（比率の正準）と `src/genome/generator.js`
（男女バイアスとレンジ）に実装。`tools/render.mjs` で目視検証しながら調整した。

## 0. 様式の核（チャンキー・セルシェード フラットイラスト）
参考（Figma/Shutterstock の 2D イラストアバター系）に酷似させた様式。`face.js` で合成。
- **円形ディスク構図**: 鮮やかな単色の円（`BG_COLORS`）の上に頭＋肩のバストを置き、
  円でクリップ。バストは 300×380 のネイティブ顔空間で組み、`translate+scale(0.82)` で円内に配置。
- **輪郭線なし・2トーンのセルシェード**:
  - 髪 = ベース色＋**クレセント影**（cap をシフトして下右に影を残す）＋クラウンの艶。
  - 肌 = 片側に薄い影ウェッジ（head にクリップ）。
  - 影色は純黒でなく暖色寄り（`color.js` の `shade`/`mix`）。
- **目は小さくシンプル**: 濃い虹彩＋下方の弱い光＋1つのハイライト。眉ははっきり。
- **鼻は一筆の控えめなチップ**、**口は穏やかな笑み**（`smileOpen` で歯見せグリン）。
- 付帯: 眼鏡（丸フレーム）、ヒゲ、頬の血色、服＋襟。

## 1. 比率の正準（front view canon）
座標系 `viewBox 0 0 300 380`、`cx=150`。頭の高さ `H = chinY - topY`（既定 268）。

| 指標 | 位置 | 根拠 |
|---|---|---|
| アイライン | 頭の縦中央（topY + 0.52H） | Loomis: 目は頭の上下中央 |
| 顔幅 | ≒ 0.71H（cheekRatio 0.355×2） | 頭幅は高さの約2/3 |
| 目幅 | 顔幅 / 5 | 「顔は目5つ分」=目の間は目1つ分 |
| 目の間隔 | 目幅1つ分（eyeGapEyes=1.0） | 同上 |
| 眉 | アイライン − browGap | 目のすぐ上 |
| 鼻先 | アイラインと顎の中間（noseYRatio 0.70） | canon |
| 口 | 鼻先→顎の約1/3（mouthYRatio 0.85） | canon |
| 耳 | 眉〜鼻先の高さに収める | canon |

silhouette は「上頭蓋＝楕円弧 → 頬（最大幅 widestYRatio 0.46）→ 顎角 jawY →
顎先 chin」を Bézier で接続（`parts.headPath`）。

## 2. 男女の差分（fem 軸 0=男性 / 1=女性、中間=中性）
`generate(seed, {fem})` の `fem` で全パラメータを線形補間。相関を保つため
個別ランダムではなく**1本の軸**で制御する。

| 部位 | 男性寄り | 女性寄り |
|---|---|---|
| 顎/輪郭 | 広い・角張る（jawRatio↑ chinRatio↑） | 狭い・顎先シャープ |
| 目 | 小さめ・細い | 大きい・高い・外側上がり＋まつ毛 |
| 眉 | 太い・低い・直線的 | 細い・高い・緩いアーチ |
| 鼻 | 広い | 細い |
| 口/唇 | 薄い | ふっくら（lipFull↑） |
| 首 | 太い | 細い |
| 髪 | shortMale / sidePart | bangsStraight / sidePart / bangsRound |

## 2.5 年齢の差分（age 18→52、ageT=clamp((age-20)/30)）
`generate(seed, {age})` の age（年）で 10〜20代〜50代を表現。`generator.js` が ageT で補間。
- **比率**: 目線を中央へ（幼さ↓ `eyeYRatio 0.56→0.50`）、目をやや小さく、顎を少し重く、鼻幅↑、唇のふくらみ↓。
- **シワ** `parts.ageLines(P,s,age)`（skinShadow の薄いストローク）: ほうれい線(28+)、目の下(34+)、
  額の横ジワ(40+/48+で2本)、目尻のカラスの足跡(43+)。intensity は ageT で増す。
- **髪**: 白髪化確率↑（`GRAY_HAIR`、男性は40〜、女性は44〜）、男性は生え際後退（`recede`→hairのfringe/templeを上げる）。
- **その他**: 血色（blush）↓、眼鏡確率↑。

## 3. 崩れ防止の実装ルール（過去に踏んだ地雷）
- **髪は2トーンにしない**: 前髪=`hair`、後ろ髪=`shade(hair,0.16)` の同系色。
  別色の暗い塊にすると「フード」に見える。
- **生え際を水平直線にしない**: 額を横切る直線は「水泳帽」化する。
  Catmull-Rom（`smoothClosed`）で、こめかみコーナー→生え際を滑らかに繋ぐ。
- **こめかみのトゲ防止**: 側面アンカーを頭頂に置かず、`templeX/templeDepth` の
  中間コーナーを必ず挟む（側面の落ち込み→生え際を単調にする）。
- **鼻は描き込みすぎない**: 中央縦ラインは折り目に見える。片側の短いチップ影＋
  薄い鼻孔のみ。
- **耳は控えめ＋ロングヘアでは後ろ髪で隠す**（z順: ears → hair.back → head）。
- 影色は純黒でなく暖色寄り（`#3a2b2b`）に寄せて mix する（フラットでも濁らない）。

## 4. 検証ループ
`node tools/render.mjs` → `.preview/sheet.png` に男女アンカー＋ランダムシードの
コンタクトシートを生成。新しいパーツ/レンジは必ずこのシートで目視してから確定する。
